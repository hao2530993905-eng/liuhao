# SCREW07091444 项目说明

## 当前定位

本项目是一个分层验证中的自动螺丝装配研究项目，目标硬件为 **UR5e 六轴机械臂、Damiao DM4310 无刷电机电批和相机**，数据采集与后续策略学习使用 LeRobot。

当前可运行能力分为两条流程：

```text
基础运动 Demo：人工提供孔前位姿 -> moveJ 接近 -> 键盘工具坐标 XY 对准
              -> 沿工具 Z 轴 moveL 低速预插 -> 记录 TCP 力数据

LeRobot 采集流程：人工/相机提供目标 -> 记录机器人状态和图像/电批状态
                 -> 确认后调用 DM4310 服务 -> 保存 LeRobotDataset
```

基础运动 Demo 不发送电批旋转命令；LeRobot 记录流程会调用电批服务。视觉自动定位、力控、扭矩闭环、螺丝拧紧完成判定和模仿学习部署尚未形成完整闭环。

## 目录职责

```text

apps/camera/           相机采集、ring 检测、点云拟合、常驻 HTTP 服务
apps/screw_driver/     DM4310/OpenArm CAN C++ 服务端和监控工具
lerobot_robot_screw/   LeRobot 包、数据特征、采集入口和数据工具
scripts/benchmarks/    启动性能实验
artifacts/             数据集、日志、相机结果、模型等运行产物
archive/legacy/        旧版和重复入口，仅用于历史追溯
third_party/           外部依赖，不在本项目重构范围内
```

源码文件开头包含统一项目文件头，运行产物统一写入 `artifacts/`，不应再写回源码目录。

## 推荐入口

先运行：

```bash
python apps/screw_demo/main.py --help
```

基础运动流程：

```bash
python apps/screw_demo/main.py \
  --enable-robot \
  --robot-host 192.168.1.5 \
  --approach-pose X Y Z RX RY RZ \
  --insert-depth 0.002 \
  --insert-speed 0.003 \
  --insert-sign 1 \
  --force-log-dir artifacts/logs/screw_demo
```

这里的 XYZ 是米，旋转向量是弧度；按键交互和状态机说明见根目录 `README.md`。

LeRobot 数据采集流程：

```bash
python lerobot_robot_screw/scripts/recording/record_main_insert_dataset.py \
  --enable-robot --robot-host 192.168.1.5 \
  --approach-pose X_MM Y_MM Z_MM RX RY RZ \
  --insert-depth 2 --insert-speed 3 --insert-sign 1 \
  --screw-host 127.0.0.1 --screw-port 5055 --screw-speed-rpm 60 \
  --dataset-root artifacts/datasets/real_insert_dataset \
  --repo-id local/real_screw_insert_001 --fps 20
```

该记录器的 XYZ、插入深度和插入速度使用毫米、毫米每秒作为命令行单位，进入 UR5e 驱动前会转换为米。它会在确认后启动电批，使用前必须先完成独立低速测试。

## 相机入口

一次相机检测：

```bash
python apps/camera/ring.py --help
```

启动常驻服务：

```bash
python apps/camera/camera_service.py --host 127.0.0.1 --port 5060
```

相机并行数据采集：

```bash
python lerobot_robot_screw/scripts/recording/record_camera_insert_dataset_parallel.py \
  --camera-ip 192.168.1.66 \
  --target-ring outer \
  --calibration-json path/to/camera_to_robot.json \
  --camera-service-url http://127.0.0.1:5060/capture \
  -- \
  --enable-robot --robot-host 192.168.1.5 \
  --insert-depth 2 --insert-speed 3 --insert-sign 1 \
  --dataset-root artifacts/datasets/real_insert_dataset --fps 20
```

没有有效手眼标定文件时，程序中的默认接近位姿只能用于流程或性能测试，不能用于真实自动对孔。

## 电批服务

```bash
cd apps/screw_driver
cmake -S . -B build -DUSE_OPENARM_CAN=ON
cmake --build build
./build/screw_server --openarm --can can0 --send-id 0x06 --recv-id 0x16 --motor-type 1 --port 5055
```

## 运行电批

```bash
cd ~/SCREW07091444

PYTHONPATH=apps/screw_demo python -c '
import time
from screw_client import make_screw_client, assert_ok

with make_screw_client("127.0.0.1", 5055) as client:
    print(assert_ok(client.status(), "status_before"))
    print(assert_ok(client.set_speed(60), "set_speed_30"))
    time.sleep(1.0)
    print(assert_ok(client.hold(), "hold"))
    print(assert_ok(client.status(), "status_after"))
'
```

构建目录是运行时生成物，已经从项目整理范围中排除；在 Linux 机器上按实际路径执行。Socket 服务启动后，先查询状态，再做低速、短时间、无负载旋转测试。
