# SCREW07091444 项目说明

## 当前定位

本项目是一个分层验证中的自动螺丝装配研究项目，目标硬件为 **UR5e 六轴机械臂、Damiao DM4310 无刷电机电批和相机**，数据采集与后续策略学习使用 LeRobot。

当前可运行能力分为两条流程：

```text
基础运动 Demo：人工提供孔前位姿 -> moveJ 接近 -> 键盘工具坐标 XY 对准
              -> 沿工具 Z 轴 moveL 低速预插 -> 记录 TCP 力数据

LeRobot 采集流程：人工/相机提供目标 -> 记录机器人状态和图像/电批状态
                 -> 确认后调用 DM4310 服务 -> 保存 LeRobotDataset
```

基础运动 Demo 不发送电批旋转命令；LeRobot 记录流程会调用电批服务。视觉自动定位、力控、扭矩闭环、螺丝拧紧完成判定和模仿学习部署尚未形成完整闭环。

## 电批 C++ 服务

`screw_driver` 用于运行电批 C++ 控制服务，并通过 Socket 向 Python 提供命令接口。

连接can总线：

```bash
sudo ip link set can0 down
sudo ip link set can0 type can bitrate 1000000 dbitrate 5000000 fd on
sudo ip link set can0 up
```
起服务:
```bash
./build/screw_server --openarm --can can0 --send-id 0x06 --recv-id 0x16 --motor-type 1 --port 5055
```

旋转电批：
```bash
PYTHONPATH=/home/sjh/Desktop/liuhao-main/apps/screw_demo \
/home/sjh/anaconda3/envs/lerobot_eye/bin/python -c '
import time
from screw_client import make_screw_client

with make_screw_client("127.0.0.1", 5055) as client:
    print(client.status())
    print(client.set_speed(-60))
    try:
        while True:
            client.heartbeat()
            time.sleep(0.1)
    except KeyboardInterrupt:
        print(client.hold())
'
其中client.set_speed(-60)是控制旋转速度的
```
检查can总线连接：

```bash
ip addr show can0
```

模拟模式构建：

```bash
cd SCREW/screw_driver
cmake -S . -B build
cmake --build build
```

运行模拟服务：

```bash
./build/screw_server --port 5055
```

测试 Python 客户端的 dry-run：

```bash
cd SCREW/screw_demo

python tests/test_screw_client.py --dry-run
```

真实电批模式需要在 Ubuntu 工控机上完成 SocketCAN、`openarm_can` 和 C++
服务端配置。运行真实硬件前，必须确认服务端已实现：

- Python 客户端断开后的速度指令看门狗。
- 急停和故障状态下持续发送零速或禁用电机。
- 正确的 JSON 协议解析。
- CAN 反馈超时和通信故障检测。
- 速度、角度、扭矩和电机状态反馈。
